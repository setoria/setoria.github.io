# Downloads

Lege das PDF fuer den stabilen Handbuch-Link hier ab:

- `TWK_Benutzerhandbuch.pdf`

Erzeugen (interne Kapitel-Links im PDF): im Repo **twk_2025** mit aktiviertem **myvenv**  
`python utils/export_benutzerhandbuch_pdf.py --copy-to <Pfad zu dieser Datei>/TWK_Benutzerhandbuch.pdf`  
Das Skript wandelt Markdown-Links (`*.md`) in PDF-Sprungmarken um.

Dann ist es dauerhaft unter dieser URL erreichbar:

- `/downloads/TWK_Benutzerhandbuch.pdf`
